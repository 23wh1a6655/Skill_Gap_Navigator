# Skill-Gap Navigator: Personalized Learning Roadmap Generator

## Overview

Skill-Gap Navigator is a full-stack web application designed to help students and working professionals identify the gap between their current skills and the skills required for their target job roles. The system provides a personalized learning roadmap, recommended resources, quiz-based self-assessment, and progress tracking to improve career readiness and employability.

Instead of following random learning paths, users can choose a specific target role such as Frontend Developer, Java Developer, Data Analyst, or Full Stack Developer and receive a structured week-by-week learning plan.

## Modules

### 1. User Registration and Authentication

Allows users to register, log in securely, and manage access using Spring Security.

### 2. Skill Selection and Role Mapping

Users select their target job role and enter their current skills for analysis.

### 3. Skill Gap Analyzer through Resume

Analyzes the uploaded resume, detects existing skills, and compares them with required industry skills to identify missing areas.

### 4. Personalized Learning Roadmap

Generates a structured week-by-week learning plan with goals and milestones.

### 5. Quiz and Self-Assessment

Conducts quizzes to evaluate understanding and validate skill readiness.

### 6. Progress Tracking Dashboard

Displays completion percentage, readiness score, completed and pending skills.

### 7. Database Management

Stores users, skills, progress reports, quiz results, and learning history.

### 8. API Testing and Validation

Ensures backend APIs work correctly using Swagger and testing tools.
